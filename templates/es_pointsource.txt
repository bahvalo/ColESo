### Subsonically moving point source. If moving velocity is constant, then the problem is equivalent to a steady point source in a uniform flow
### f(t) parameters
#Ampl 1.0        # multiplicator
#Freq 0.05       # linear frequency i. e. omega = 2*pi*Freq
#Phase 0.0       # phase
#tmin -20.0      # source acts from tmin
#tmax 0.0        # source acts till tmax
#SignalType 6    # signal form. 1: sin(2*Pi*Freq*t); 6: (sin(2*Pi*Freq*t))^4

### Space parameters of the source
#Aterm 1.0        # L1-norm of the source
#Bterm 1.0        # impulse width. Not used unless NormalizeForm flag is unset
#NormalizeForm 1  # set by default. If zero, Aterm will be considered as maximal value of f(r), and L1-norm will be recalculated using Bterm
#CoorX 0.0        # impulse coordinates
#CoorY 0.0
#CoorZ 0.0
#NAngularPeriods 1 # solution is the signal generated by NAngularPeriods impulses rotated across Z axis

### Background flow parameters: steady uniform flow
#SoundVel 1.0
#FlowMach 0.0
### Background flow parameters: unsteady uniform flow
#UnstAmpl 0.0
#UnstFreq 1.0
#UnstPhase 0.0
